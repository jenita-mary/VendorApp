# VendorApp

A vendor management application built with Python, Tkinter, and SQLite.

## Features

- Vendor Registration
- Vendor Login
- Forgot Password
- Add Products
- Edit Products
- Remove Products
- Sell Products
- Product Inventory Tracking

## Technologies

- Python
- Tkinter
- SQLite
- OOP

## Installation

```bash
pip install email-validator tabulate python-dotenv
python main.py
```
